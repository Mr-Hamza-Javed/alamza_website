ALAMZA INDUSTRY — IMAGES / ASSETS FOLDER
=========================================

Apni saari website images ISI folder (assets/) mein rakhein.

Zaroori naam (bilkul yehi spelling, chhote harf):
  logo.png            -> nav + favicon ka logo
  hero-part.png       -> hero (sab se upar) ka bara render  [transparent PNG behtar]
  about-factory.jpg   -> About section ki factory photo

Products / Team / Gallery / Showcase ki images ke naam aap khud
site-data.json mein "img" field se tay karte hain — bas wahi file
yahan assets/ folder mein daal dein. Misal:
  site-data.json  ->  "img": "team-1.jpg"
  to file rakhein ->  assets/team-1.jpg

Tip: images 500 KB se chhoti rakhein to website tez khulegi.
